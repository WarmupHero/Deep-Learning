------
For each run, convergence epoch is the first epoch after which all remaining training-loss values stay within a small tolerance of the final training loss.

For overall loss comparison across classification and regression, combined results use normalized best validation loss within each problem separately.

Answers
-------
1. Fastest optimizer on average over all experiments: sgd
2. Best optimizer by average loss-function value over all experiments using average normalized best validation loss: adabelief 
3. How network depth affects optimization: Overall, A2 (3 hidden layers) converges faster on average, while A1 (1 hidden layer) achieves the better average normalized loss.
